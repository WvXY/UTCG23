Description

In this assignment, I decided to draw a circle using NURBS(Non-Uniform Rational B-Spline).
When I was making architectural design, I often used a software called Rhinoceros, which uses NURBS to make 3D models. It can draw some very precise and smooth surfaces, compared with other software such as SketchUp and C4D, which impressed me a lot at that time. So, that is why I decided to draw NURBS in this work.

Python Ver. also available on Google Colab

As you can see, so far there are some problems. One problem is, it doesn't support editing, so you can't change the parameters here. In order to draw something different, (e.g, change/add more control points, change weights, etc.) you have to edit the source code, which is to some extent, not user-friendly.
Another problem is that I am an amateur. I don't know how to draw 2 objects in one canvas using WebGL. I might fix it in the next update.
