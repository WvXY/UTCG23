Assignment 1: Spline

Description

In this assignment, I decided to draw a circle using NURBS(Non-Uniform Rational B-Spline).
When I was doing architecture design, I often used a 3D modeling software called Rhinoceros, which uses NURBS to make precise 3D models.

The problem is, it doesn't a GUI, so it can't be changed in this page. In order to draw something different, (e.g, add more control points, change weights, etc.) you have to edit the source code, which is to some extent, not user-friendly.
